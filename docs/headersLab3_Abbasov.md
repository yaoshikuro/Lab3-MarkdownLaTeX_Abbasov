<h2>Лабораторная работа №3. Работа с Markdown-разметкой и базовое использование LaTeX в документации проекта </h2>
<h4>Markdown — это облегчённый язык разметки для форматирования текста простыми символами.</h4>
<h4>Markdown применяется везде, где нужно быстро и красиво оформить текст, не отвлекаясь на кнопки форматирования.</h4>
<h4>Разработчикам Markdown нужен, чтобы отделить содержание от оформления и автоматизировать создание документации.</h4>
